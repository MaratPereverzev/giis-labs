# Lab 1: Построение отрезков - Инструкция

## Как использовать

### Запуск приложения

```bash
cd /home/marat/Documents/BSUIR/GIIS/giis-labs
npm run dev
```

Откроется на http://localhost:5174

### Использование Lab 1

1. **На главной странице:**
   - Одна карточка "Лаб 1 - Построение отрезков"
   - Нажмите на карточку для входа

2. **В Lab 1:**
   - Выберите алгоритм (кнопки слева):
     - DDA
     - Брезенхем
     - Ву (сглаживание)

3. **Рисование линии:**
   - Нажмите на canvas для установки начальной точки (зеленая)
   - Нажмите еще раз для конечной точки (красная)
   - Между ними автоматически рисуется линия черным цветом

4. **Дополнительно:**
   - Кнопка "Очистить" удаляет линию
   - Можно менять алгоритм и сравнивать результаты
   - Координаты точек отображаются в левой панели

## Алгоритмы

### DDA (Digital Differential Analyzer)
- Использует дифференциальные уравнения
- Хорошо подходит для обучения
- Медленнее чем Брезенхем

### Брезенхем
- Целочисленный алгоритм
- Быстрый и эффективный
- Самый используемый в практике

### Ву (антиалиасинг)
- Применяет сглаживание
- Гладкие линии на косых углах
- Медленнее других

## Алгоритмы в коде

### Использование

```tsx
import { drawLineDDA, drawLineBresenham, drawLineWu } from '../utils/lineDrawing';

// DDA
const points = drawLineDDA(x0, y0, x1, y1);

// Брезенхем
const points = drawLineBresenham(x0, y0, x1, y1);

// Ву
const points = drawLineWu(x0, y0, x1, y1);
// points содержат свойство intensity для каждой точки
```

## Разработка

### Добавить новый алгоритм

1. Добавьте функцию в `src/utils/lineDrawing.ts`
2. Экспортируйте её
3. Используйте в `src/labs/lab1/Lab1.tsx`

### Добавить новый элемент управления

1. Отредактируйте `src/labs/lab1/Lab1.tsx`
2. Добавьте JSX в метод render
3. Добавьте CSS в `src/labs/lab1/Lab1.css`
