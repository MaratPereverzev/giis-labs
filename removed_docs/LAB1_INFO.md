### Функциональность Lab 1:

1. **Выбор алгоритма:** DDA, Брезенхем или Ву
2. **Интерактивный рисунок:**
   - Нажимаете на canvas → устанавливается начальная точка (зеленая)
   - Нажимаете еще раз → устанавливается конечная точка (красная)
   - Между ними рисуется линия выбранным алгоритмом (черная)
3. **Отображение информации:**
   - Текущие координаты точек
   - Описание активного алгоритма
4. **Очистка:** Кнопка для удаления и нового рисунка

### Алгоритмы:

- **DDA**: Простой, основан на дифференциальных уравнениях. Работает медленнее, но понятен концептуально.
- **Брезенхем**: Быстрый целочисленный алгоритм. Самый популярный для растеризации линий.
- **Ву**: Применяет антиалиасинг для гладкого отображения линий. Медленнее других, но качественнее на косых линиях.
